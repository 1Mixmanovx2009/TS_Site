TypeScript haqida

TypeScript - bu JavaScriptning typed supersetidir va oddiy JavaScriptga kompilyatsiya qilinadi.

JavaScript eng mashhur dasturlash tillaridan biridir. Buning sabablaridan biri uning dinamik tarzda yozilganligidir. Boshqacha qilib aytganda, JavaScript erkinroq tildir, lekin siz loyihangizda juda ko‘p kodga duch kelsangiz, bu erkinlik vahimaga aylanishi mumkin.

Keling ushbu holatga nazar tashlab ko‘ramiz:

const pikachu = (a, b) => { return a + b;}

Biror narsa deyish qiyin, chunki pikachu funksiyasi har qanday turdagi qiymatlarni qabul qilishi mumkin, masalan, satr, raqam va hokazo.

// Raqam uzatiladi. const pikachu = (a, b) => { return a + b;}const result = pikachu(2021, 9);console.log(result);
// 2030
// Satr uzatiladi. const pikachu = (a, b) => { return a + b;}const result = pikachu("2021", "9");console.log(result);
// 20219

Misollar

function greet(name: string): string {

return `Salom, ${name}!`;
}
console.log(greet("Khojiakbar")); // Salom, Khojiakbar!

interface User {
id: number;
name: string;
isActive: boolean;
}

const user: User = {
id: 1,
name: "Khojiakbar",
isActive: true,
};

console.log(user); // { id: 1, name: 'Khojiakbar', isActive: true }

function getArray(items: T[]): T[] {
return new Array().concat(items);
}

const numberArray = getArray([1, 2, 3]);
const stringArray = getArray(["a", "b", "c"]);

console.log(numberArray); // [1, 2, 3]
console.log(stringArray); // ["a", "b", "c"]
